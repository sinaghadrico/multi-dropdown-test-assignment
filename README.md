## Installation

**Clone the repository:**
   ```bash
   git clone git@github.com:sinaghadrico/multi-dropdown-test-assignment.git
   cd multi-dropdown-test-assignment
 ```

 **Install packages:**
   ```bash
  yarn install
 ```
 **Run project:**
   ```bash
  yarn dev
 ```
  **Run tests:**
   ```bash
  yarn test
 ```
